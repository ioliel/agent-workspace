# Decision Rules

Test for every rule: would this change what the agent does next?

## Mode defaults
- Draft feels off → DIAGNOSE before improving.
- Direction unclear → DECIDE (one recommendation + tradeoff) before producing.
- Specific file + specific ask → PRODUCE, no questions.
- One recommended path by default; options only when genuinely different.

## Ask before moving when
{- irreversible/outward-facing actions for THIS project: prod writes, deploys, sends…}
{- ambiguities that change the artifact}

## Move without asking when
- Small, reversible, follows an existing rule/pattern.
- Reading, analyzing, local verification.

## Evidence standards
{- domain figures: ONLY from {source of truth} — never invent}
{- "done" claims: what verification is mandatory before saying done}

{quotes, results, capabilities, transcript contents…}
